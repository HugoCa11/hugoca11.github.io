Swarm 2

In SWARM 2, every mission presents unique challenges as you navigate the neon-lit ruins of Earth, a constantly changing battleground. As you progress, you’ll grow stronger with the help of Auggie and revolutionary Shard Tech, which allows you to permanently upgrade your skills. With dozens of perks to choose from, you’ll customize your abilities to turn the tide of war. The game features signature smooth grappling mechanics that let you swing dynamically through the environment, building momentum to outmaneuver enemies. With unexpected challenges and evolving tactics, SWARM 2 is a high-stakes, thrilling experience that tests your skills and strategic thinking.

My main tasks in this project consist on:

  • Developed and integrated features into VR games using Unity, focusing on UI and gameplay logic.
  • Unity Development: Integrated UI elements and game features based on design mockups, performed bug fixing.
  • VR Functionality: Integrated core VR features using C# to enhance gameplay experience.
  • Tools & Technologies: Worked with C#, Meta XR SDK, Fusion, GitHub, Jira, and Photoshop.
  • Collaboration: Worked closely with designers and developers for seamless UI integration.
  • Additional Contributions: Assisted in integrating VFX and implementing game logic.
  • Freelance Work: Started working on Swarm 2 as a freelancer via Upwork, contributing to development and bug fixes.
