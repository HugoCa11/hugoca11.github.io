Greensky Game Studios

In the development of Swarm 2 and Chronostrike, I played a key role in integrating various features to enhance the overall gaming experience. Working with Unity, I was tasked with implementing UI elements and gameplay logic based on design mockups provided by the UI and design teams. This involved translating static designs into functional user interfaces that were responsive and interactive within a virtual reality environment.
A significant part of my responsibilities included using C# to code the necessary game logic, while I also worked with tools like Meta XR SDK to ensure compatibility across various VR platforms, including Meta devices. Additionally, I utilized Fusion for multiplayer functionality and Photoshop for visual assets where needed. The development process was supported by GitHub for version control and Jira to track project progress, bugs, and tasks.
Collaboration was a major part of the project; I worked closely with other developers and designers to ensure that the UI elements were integrated seamlessly into the game, maintaining both functionality and aesthetic consistency.
