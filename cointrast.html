Cointrast

This project was created for the Data Visualization class in the VR/AR Design and Development program at Vancouver Film School. Using Lens Studio, we developed an AR Marker experience that presents two contrasting perspectives on how the COVID-19 pandemic affected the environment in Canada.

The first marker highlights the positive effects of the pandemic on nature, such as reduced pollution, cleaner air, and the resurgence of wildlife in urban areas. This side of the experience illustrates how the slowdown in human activity temporarily benefited the environment.

The second marker focuses on the negative consequences, including the surge in medical waste, pollution from disposable masks, and increased plastic consumption due to health concerns. It serves as a reminder that while some aspects of the environment improved, new ecological challenges emerged.

My main contributions to the project included:

  • Setting up and configuring the Lens Studio project.
  • Programming the interactive functionality for both markers using JavaScript.
  • Assembling the project and ensuring smooth integration within Lens Studio.
