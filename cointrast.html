Cointrast

This is a AR Marker developed using Lens Studio for the Data Vizualization class at the VR/AR Design and Development program at the Vancouver Film School.

The objective is to show the negative and positive effects that the COVID-19 pandemic left on the environment, specifically in Canada. For this we decided to create two different markers in order to present "both sides of the coin".

On this project my main tasks consisted on:

  • Configured the Lens Studio project
  • Programed the functionality for both markers using JavaScript
  • Assambled the project in Lens Studio
